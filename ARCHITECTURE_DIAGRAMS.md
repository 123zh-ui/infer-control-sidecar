# 架构图多格式文件说明

已生成以下文件：

- `ARCHITECTURE.puml`（PlantUML）
- `ARCHITECTURE.dot`（Graphviz DOT）
- `ARCHITECTURE.d2`（D2）
- `LOGIC_DIAGRAM.md`（Mermaid）
- `SEQUENCE_DIAGRAM.puml`（PlantUML 时序图：启动 + 请求）
- `SEQUENCE_DIAGRAM.md`（Mermaid 时序图：启动 + 请求）

## VS Code 预览建议

1. PlantUML
- 安装插件：`jebbs.plantuml`
- 打开 `ARCHITECTURE.puml` 后，执行 `PlantUML: Preview Current Diagram`

2. Graphviz DOT
- 安装插件：`joaompinto.vscode-graphviz`
- 打开 `ARCHITECTURE.dot` 直接预览

3. D2
- 安装插件：`terrastruct.d2`
- 打开 `ARCHITECTURE.d2` 进入预览

4. Mermaid（已可用）
- 打开 `LOGIC_DIAGRAM.md`
- `Ctrl+Shift+V` 打开 Markdown 预览
